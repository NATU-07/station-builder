# さびれた駅 (Station Builder)

さびれた無人駅を掃除・修理・設備追加して、駅ビルまで成長させる育成ゲーム。
HTML + CSS + JavaScript（フレームワーク無し）、Canvas API で水彩画・絵本風の描画。

---

## 1. どこに何があるか（ファイル構成）

### ソースコード（`src/`）
| ファイル | 役割 |
|---|---|
| `game.js` | ゲームループ・初期化・全体の接続 |
| `station.js` | 駅の状態（きれい度・評判・所持金・段階）とアップグレード管理 |
| `ui.js` | タブ・モーダル・ボタン・警告バナーなどDOM操作 |
| `renderer.js` | 水彩画風の背景描画 |
| `renderer-stages.js` | Stage別の駅画像差し替え |
| `cleaning.js` | 通常の掃除ミニゲーム（雑草・ほこり・ゴミ） |
| `specialDirt.js` | 特殊汚れの発生・管理（落書き・油汚れ・ゴミの山） |
| `specialDirtGames.js` | 特殊汚れのミニゲーム |
| `events.js` | ランダムイベント発火 |
| `effects.js` | フロート文字・パーティクルなどエフェクト |
| `sound.js` | 効果音再生 |
| `achievements.js` | 実績管理 |
| `journal.js` | 日記・ログ |
| `tutorial.js` | チュートリアル進行 |

### データ（`data/`）
- `items.json` — 設置アイテム定義
- `events.json` — イベント定義
- `upgrades.json` — アップグレード定義

### ドキュメント（`docs/`）— 作業時に必ず同期更新する
- **`game-data-list.md`** — アイテム・イベント・アップグレードのマスター一覧（人間向け）
- **`events-list.md`** — イベント詳細説明

### 素材
- `assets/` — Stage画像・アイコン

### エントリーポイント
- `index.html` — HTML＋CSS＋スクリプト読み込み

## 2. ゲーム仕様

- 10段階進化（Stage 0〜9）：無人駅 → 小さな駅 → … → 伝説の駅
- クリックで掃除・修理・設備追加
- 乗客数に応じて自動でお金が貯まる
- アップグレードを全部揃えると次の段階へ
- Stage 5+ で特殊汚れシステム解放（落書き・油汚れ・ゴミの山）

## 3. 開発ルール（作業中に守ること）

### コーディング
- **関数は50行以下**
- **1ファイル800行以下**（超えたら分割する）
- エラーは明示的にハンドリングする
- 変数・関数名は意味のわかる日本語風 or 英語で統一

### データ管理（重要）
- `data/items.json` / `events.json` / `upgrades.json` を変更したら **必ず `docs/game-data-list.md` も同時に更新する**
  - 該当表の行だけでなく **統計サマリー（巻末の件数表）も更新する**
- `events.json` を変更したら **`docs/events-list.md` も更新する**
- 「JSONだけ変えてmdを放置」は禁止（ユーザーがひと目で把握するためのリスト）

### UI・操作の地雷（過去にハマったやつ）
- **ミニゲーム（Promise）は必ず ✕ボタン／ESC で中断できる道を用意する**
  - 「全クリア」だけが終了条件だと操作不能（stuck）になる
- **ゲームループから `innerHTML` 再構築しない**
  - mousedown → mouseup の間に要素が差し替わるとクリックが発火しない
  - モーダル等は「件数が変わった時だけ」再描画する
- **Canvas で絵文字を `fillText` する前に `fillStyle` を不透明色に戻す**
  - 直前のfillStyleが半透明だと、絵文字も半透明で描画されてしまう（影→絵文字の順に描く時は要注意）
- **新規設置アイテムは、自動手入れONなら `selectedIds` に自動追加する**
  - 設置した瞬間から劣化対象に入るため、手入れ対象からの取り残しを防ぐ

### バグチェック
- 機能を追加・変更したら **`code-reviewer` エージェントでバグチェックを実行**
- 見つかったバグは自分で修正してからユーザーに報告する

### 禁止事項
- ユーザーが指定したファイル以外を削除しない
- 作業範囲は `D:\ゲーム制作\station-builder\` 内のみ
