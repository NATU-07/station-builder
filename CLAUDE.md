# さびれた駅 (Station Builder)

## 目的
さびれた無人駅を掃除・修理・設備追加して、駅ビルまで成長させる育成ゲーム。

## 禁止事項
- ユーザーが指定したファイル以外を削除しないこと
- 作業範囲は `D:\ゲーム制作\station-builder\` 内のみ

## 技術スタック
- HTML + CSS + JavaScript（フレームワークなし）
- Canvas API で水彩画風の描画
- ブラウザゲーム

## ファイル構成
- `index.html` — エントリーポイント
- `src/game.js` — ゲームロジック
- `src/station.js` — 駅の段階・アップグレード管理
- `src/renderer.js` — 水彩画風の描画
- `src/ui.js` — UI・クリック操作
- `data/upgrades.json` — アップグレード定義
- `assets/` — 画像素材

## ゲーム仕様
- 5段階進化：無人駅 → 小さな駅 → 中規模駅 → 大きな駅 → 駅ビル
- クリックで掃除・修理・設備追加
- リアルタイムでお金がたまる（乗客数に応じて）
- アップグレードを全部揃えると次の段階へ
- 水彩画・絵本風の見た目

## バグ管理ルール
- 機能を追加・変更したら、マルチエージェント（code-reviewer）でバグチェックを実行する
- 見つかったバグは自分で修正してからユーザーに報告する

## コーディングルール
- 関数は50行以下
- ファイルは800行以下
- エラーは明示的にハンドリング
