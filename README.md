# さびれた駅

さびれた無人駅を掃除・修理して、伝説の駅まで育てる放置＆育成ゲームです。

## 遊ぶ

ブラウザで開くだけで遊べます。
PCブラウザ推奨（Chrome / Edge / Firefox）。

## 特徴

- ワンクリック操作の駅育成ゲーム
- 10段階の進化（さびれた無人駅 → 伝説の駅）
- お掃除ミニゲーム
- 自動手入れ機能（Stage 1 以降、アイテム5個以上設置で解放）
- ランダムイベント（24種）
- 実績システム
- 11段階の評判ランク
- オフライン収入（12時間ぶんまで）

## 技術

- HTML + CSS + JavaScript（フレームワーク不使用）
- Canvas API で描画
- localStorage でセーブ

## ライセンス

個人制作のゲームです。
