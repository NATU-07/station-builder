# イベント一覧（全37種）

## タイプ説明

| タイプ | 説明 |
|--------|------|
| choice | プレイヤーが選択肢を選ぶ。結果が変わる |
| instant | 即時発動。お金・評判・きれい度が変化 |
| buff | 一定時間、乗客数や収入に倍率がかかる（デバフ含む） |
| conditional | 条件を満たしているかで結果が変わる |

## バフ/デバフ効果一覧

| 効果キー | 説明 | 値の例 |
|----------|------|--------|
| passengerMult | 乗客数倍率 | x2.0=2倍、x0.3=7割減 |
| incomeMult | 収入倍率 | x1.5=1.5倍、x0.3=7割減 |
| cleanPowerMult | 掃除効率倍率 | x0.5=半分しか掃除できない |
| reputationGainMult | 評判獲得率倍率 | x0.3=評判が上がりにくい |
| durabilityDamagePercent | 全アイテム耐久即時ダメージ | 50=全アイテム耐久半減 |
